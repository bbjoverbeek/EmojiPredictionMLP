# EmojiPredictionMLP
EmojiPrediction project for MLP

## Data
Dataset is from [Hugging Face](https://huggingface.co/datasets/tweet_eval)

## Running the program
Create virtual env:

`$ python3 -m venv env`

`$ source env/bin/activate`

`$ python3 -m pip install -r requirements.txt`

Run the program: 

`$ python3 main.py`
